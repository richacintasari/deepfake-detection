# 🚀 Deepfake Detection

## 📌 Deskripsi Project

Aplikasi berbasis web yang digunakan untuk mendeteksi apakah suatu media (gambar/video) merupakan **deepfake** atau asli (real).
Sistem ini dibuat sebagai implementasi dari perancangan pada *Daily Project 6* dan dikembangkan pada *Daily Project 7*.

Aplikasi memungkinkan pengguna untuk:

* Mengunggah file media
* Melakukan proses deteksi
* Melihat hasil berupa status (REAL / FAKE)
* Menampilkan confidence score dari hasil analisis

---

## 🎯 Fitur Utama

* Upload file (image/video)
* Deteksi deepfake berbasis AI (simulasi)
* Menampilkan hasil REAL / FAKE
* Confidence score (%)
* Preview file sebelum upload
* UI modern dan responsif

## ⚙️ Cara Menjalankan Project

### 1. Clone Repository

```
git clone https://github.com/username/deepfake-detection.git
cd deepfake-detection
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Jalankan Aplikasi

```
python app.py
```

### 4. Akses di Browser

```
http://127.0.0.1:5000
```
